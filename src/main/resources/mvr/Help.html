All functionality is available by right-clicking on the views you want to process. Simply select one or more views in the ViewSetup Explorer table, then right-click to access all processing options.

For macro recording: Please use the Fiji BigStitcher plugins menu instead of the interactive explorer interface, as the explorer does not support macro recording.

Key Bindings for Multiview-Reconstruction

When the BigDataViewer window is active (right click), it will display those image stacks that are currently selected in the ViewSetup Explorer. While the ViewSetup Explorer is in focus, you can use special keys to change the way the images are displayed.

When the BigDataViewer window is active, it has special key-bindings that allow you to browse and display the data. Please press F1 then to display another help window with more details.
